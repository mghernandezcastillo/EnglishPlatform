# English Class Curriculum Template

To ensure all classes have a consistent, engaging, and rich experience for the students, follow this strict 17-slide structure designed for a **1-hour class**.

## Structure (5 Sections)

### 1. Warm-up (5 mins) - 1 Slide
- **Slide 1:** Welcome! A quick question or engaging picture to break the ice and get the student speaking or reviewing a previous concept.
- **Visuals:** Always include a bright, welcoming image (`imageUrl`).
- **Colors:** Blue or bright gradients (`bg-gradient-to-br from-cyan-500 to-blue-600`).

### 2. Rules, Vocabulary & Grammar (15 mins) - 4 Slides
- **Slide 2:** Intro to the main topic or grammatical rule.
- **Slide 3:** Examples and formulas.
- **Slide 4:** More examples, exceptions, or deep dive into the grammar.
- **Slide 5:** New Verbs / Action words / Vocabulary list. Always include a relevant image.
- **Colors:** Teal, Indigo, or Emerald gradients.

### 3. Practice & Reading (15 mins) - 5 Slides
- **Slide 6:** Reading Practice. A short paragraph reading exercise to practice pronunciation. Include an engaging image.
- **Slide 7:** Multiple choice / Complete the sentence (1/3). Use `options` and `correctOptionIndex`.
- **Slide 8:** Multiple choice / Complete the sentence (2/3). 
- **Slide 9:** Multiple choice / Complete the sentence (3/3).
- **Slide 10:** Find the mistake. Provide 3 incorrect sentences for the student to fix verbally.
- **Colors:** Orange, Amber, or Red gradients for alert/focus.

### 4. Production & Quiz (15 mins) - 5 Slides
- **Slide 11:** Describe the picture / Roleplay / Conversation prompt. Get the student talking mostly unguided. Add an image.
- **Slide 12:** Fun Quiz! (1/4). Multiple choice to check retention.
- **Slide 13:** Fun Quiz! (2/4). Multiple choice to check retention.
- **Slide 14:** Fun Quiz! (3/4). Multiple choice to check retention.
- **Slide 15:** Fun Quiz! (4/4). Vocabulary or spelling check.
- **Colors:** Green or Pink/Rose gradients for energy.

### 5. Wrap-up (10 mins) - 2 Slides
- **Slide 16:** Great Job Today! Summary of accomplishments. A congratulatory image.
- **Slide 17:** Homework. Clear assignment for the next class.
- **Colors:** Violet or Slate/Dark gradients for closure.

## Developer Rules
- Always map `options` string array and `correctOptionIndex` number for interactive multiple-choice.
- Provide `imageUrl` (Unsplash) for Slides 1, 3/4/5 (vocabulary), 6 (reading), 11 (picture description), and 16 (congratulations).
- Stick to 17 slides minimum for a 1-hour session.
- Let the `bgColor` create distinct chunks in the presentation visually.
